Number Spiral

Rating: 1600

In a number spiral, numbers are arranged in a square spiral pattern. Given a position (y, x), find the number at that cell.

Input

The first line contains an integer t (number of queries). Each of the next t lines contains two integers y and x.

Output

For each query, print the value at coordinate (y, x) in the number spiral.
